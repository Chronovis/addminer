# Addminer

## Setup
    $ git clone https://github.com/Chronovis/addminer.git
    $ npm i
    
## Develop

### Run    
    $ npm run server
    $ npm run client-server
    
### Watch client
    $ npm run watch-client
    $ npm run watch-stylus
    
### Watch server    
    $ npm run watch-server
